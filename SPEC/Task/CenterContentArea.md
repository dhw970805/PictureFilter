# 中央内容区详细实现
## 1. 视图模式
视图模式分为网格视图，列表视图，详情视图， 视图模式的背景色为深色。
### 1.1 网格视图
- 缩略图显示（间距10px，默认尺寸150×150px）
- 文件信息显示（缩略图+文件名）
- 选中态效果（2px边框 #4A9EFF）
- 双击打开文件功能（仅实现信号发送）
- 右键操作菜单（未实现具体菜单内容）
- 网格视图根据预览尺寸的数值，从左到右进行自适应排列，超出一行时，在新的一行排列图片，不应当超出左右布局边界，超出布局上下边界时显示上下滑动的滚动条

### 1.2列表视图
- 列表项显示（缩略图、名称、大小、修改日期、类型、分辨率）
- 排序功能（点击列标题升序/降序）
- 选中列标题高亮

### 1.3 详情视图
- 左侧大尺寸预览（400×自适应）
- 右侧文件基本信息和元数据
