
# 首界面设计规格

## 文档信息

## 一、核心设计原则

1. **高效性**：优先满足媒体文件（图片/视频）的快速浏览、筛选、管理，操作路径最短化；

2. **一致性**：界面组件样式、交互逻辑对齐 Adobe Bridge 核心体验，降低用户学习成本；

3. **可定制性**：支持面板布局调整、视图切换、快捷键自定义；

4. **性能优先**：界面渲染不阻塞文件加载/预览，保证大文件（如RAW/4K视频）流畅预览。

## 二、整体界面架构

### 2.1 界面布局（参考Adobe Bridge经典布局）

整体采用「模块化可拆分面板」设计，分为 5 个核心区域，支持面板拖拽调整/隐藏/重置：

```Plain Text

┌─────────────────────────────────────────────────────────────┐
│ 【顶部菜单栏】【快捷工具栏】                                 │
├──────────┬───────────────────────┬──────────────────────────┤
│ 【左侧面板】│ 【中央内容区】       │ 【右侧属性面板】          │
│          │                       │                          │
│          │                       │                          │
│          │                       │                          │
├──────────┴───────────────────────┴──────────────────────────┤
│ 【底部状态栏】                                               │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 各区域详细规格

#### 2.2.1 顶部菜单栏（固定不可隐藏）

- **尺寸**：高度 32px（Windows）/ 36px（macOS），宽度自适应窗口；

- **结构**：

|    菜单项|    子菜单核心功能|
|---|---|
|    文件(F)|    新建文件夹、导入、导出、批量重命名|
|    编辑(E)|    复制/粘贴、批量修改、偏好设置|
|    视图(V)|    视图切换（列表/网格/详情）、缩放|
|    筛选(S)|    按类型/日期/大小/标签筛选|
|    工具(T)|    批量处理、元数据编辑、插件管理|
|    帮助(H)|    教程、关于、检查更新|
- **样式**：

    - 字体：Windows 用「微软雅黑 12px」，macOS 用「SF Pro 12px」；

    - 选中态：背景色 `#0078D4`（Windows）/ `#0066CC`（macOS），文字白色；

    - 悬停态：背景色 `#E8F0FE`（Windows）/ `#F0F7FF`（macOS）。

#### 2.2.2 快捷工具栏（可隐藏）

- **尺寸**：高度 40px，宽度自适应，位于菜单栏下方；

- **核心按钮**（从左到右）：

|    按钮|    图标样式|    快捷键|    功能说明|
|---|---|---|---|
|    返回|    ←（Adobe风格线性图标）|    Alt+←|    返回上一文件夹|
|    前进|    →（同上）|    Alt+→|    前进下一文件夹|
|    刷新|    🔄（同上）|    F5|    刷新当前目录|
|    视图切换|    网格/列表图标切换|    Ctrl+1/Ctrl+2|    切换网格/列表视图|
|    排序|    ↓↑ 箭头|    Ctrl+S|    打开排序选项（名称/日期/大小）|
|    筛选|    🔍+筛选图标|    Ctrl+F|    打开筛选面板|
|    批量处理|    批量图标|    Ctrl+B|    打开批量操作窗口|
- **样式**：

    - 按钮尺寸：32×32px，圆角 4px；

    - 常态：透明背景，图标色 `#333333`；

    - 悬停：背景色 `#E0E0E0`，图标色 `#000000`；

    - 点击：背景色 `#D0D0D0`，图标色 `#0078D4`。

#### 2.2.3 左侧面板（可拖拽调整宽度/隐藏/拆分）

- **默认尺寸**：宽度 200px，高度自适应；

- **核心模块**（可折叠）：

    1. **导航栏**：

        - 快速访问：桌面、文档、图片文件夹；

        - 最近访问：显示最近打开的 10 个目录；

        - 收藏夹：支持拖拽添加/右键删除目录；

    2. **筛选器**：

        - 按文件类型：图片（JPG/PNG/RAW）、视频（MP4/MOV）、音频；

        - 按日期：今天、昨天、本周、本月；

        - 按标签：红色/黄色/绿色/自定义标签；

    3. **预览尺寸**：

        - 滑块调整中央内容区缩略图大小（50px ~ 500px）；

- **样式**：

    - 模块标题：字体 12px 加粗，背景色 `#F5F5F5`；

    - 列表项：悬停背景 `#E8F0FE`，选中背景 `#0078D4`（文字白色）；

    - 分割线：1px 实线 `#E0E0E0`。

#### 2.2.4 中央内容区（核心区域，不可隐藏）

- **尺寸**：自适应剩余空间；

- **核心视图模式**（支持切换）：

    1. **网格视图（默认）**：

        - 缩略图间距：10px；

        - 缩略图默认尺寸：150px×150px；

        - 显示信息：缩略图+文件名（底部居中），选中态添加 2px 边框 `#0078D4`；

        - 交互：双击打开文件、右键显示操作菜单、拖拽多选；

    2. **列表视图**：

        - 列项：缩略图（50px×50px）、名称、大小、修改日期、类型、分辨率；

        - 排序：点击列标题升序/降序，选中列标题下划线高亮；

    3. **详情视图**：

        - 左侧：大尺寸预览图（400px×自适应）；

        - 右侧：文件基本信息+元数据（EXIF/视频参数）；

- **性能要求**：

    - 支持懒加载：仅渲染可视区域内的缩略图；

    - 大文件夹（>1000文件）：先加载缩略图占位符，再异步加载实际预览图；

    - 预览缓存：缓存已加载的缩略图，重复访问不重新渲染。

#### 2.2.5 右侧属性面板（可拖拽调整宽度/隐藏）

- **默认尺寸**：宽度 300px，高度自适应；

- **核心模块**（可折叠）：

    1. **基本信息**：文件名、大小、格式、分辨率、修改日期；

    2. **元数据**：EXIF（相机型号、快门、光圈、ISO）、GPS（如有）；

    3. **标签/评分**：

        - 评分：1-5星评分控件；

        - 标签：红色/黄色/绿色/蓝色/紫色标签，支持自定义标签；

    4. **快速编辑**：

        - 旋转：顺时针/逆时针旋转；

        - 裁剪：快速裁剪预览（不修改原文件）；

        - 备注：文本框输入文件备注；

- **样式**：

    - 模块间距：16px；

    - 文本：标签色 `#666666`，值色 `#333333`，字体 12px；

    - 输入框：边框 1px `#E0E0E0`，聚焦边框 `#0078D4`。

#### 2.2.6 底部状态栏（可隐藏）

- **尺寸**：高度 24px，宽度自适应；

- **显示内容**（从左到右）：

    1. 当前路径：如「桌面 > 图片 > 旅行」；

    2. 文件统计：如「共 245 个文件，选中 12 个」；

    3. 预览模式：如「网格视图 | 缩略图大小：150px」；

    4. 加载状态：如「正在加载：35/245」（加载中显示进度）；

- **样式**：

    - 背景色 `#F5F5F5`，文字色 `#666666`，字体 11px；

    - 进度提示：文字色 `#0078D4`。

## 三、核心交互逻辑

### 3.1 面板操作

1. **拖拽调整**：面板边缘拖拽可调整宽度/高度，最小宽度/高度限制：

    - 左侧面板：最小 100px；

    - 右侧面板：最小 200px；

2. **隐藏/显示**：右键面板标题栏可隐藏，或通过「视图 > 面板」菜单恢复；

3. **重置布局**：「视图 > 重置布局」恢复默认面板位置。

### 3.2 文件操作

1. **预览**：

    - 鼠标悬停在缩略图上显示大图预览（延迟 300ms，预览图尺寸 400×400）；

    - 滚轮缩放：在中央内容区滚轮可调整缩略图大小；

2. **多选**：

    - 按住 Ctrl 点击：多选不连续文件；

    - 按住 Shift 点击：多选连续文件；

    - 拖拽框选：框选区域内文件；

3. **批量操作**：

    - 多选文件后，右键/工具栏「批量处理」可批量重命名/修改标签/导出；

    - 批量操作时显示进度条，支持取消操作。

### 3.3 快捷键规范（对齐Adobe Bridge）

|复制文件|Ctrl+C|

|删除文件|Delete|
|重命名文件|F2|
|切换网格视图|Ctrl+1|
|切换列表视图|Ctrl+2|
|切换详情视图|Ctrl+3|
|放大缩略图|Ctrl++|
|缩小缩略图|Ctrl+-|
|刷新|F5|
|打开筛选面板|Ctrl+F|

|文字次色|`#666666` / `#6E6E73`|备注、辅助文本|

### 4.2 字体规范

|平台|标题字体|正文字体|辅助字体|
|---|---|---|---|
|Windows|微软雅黑 14px 加粗|微软雅黑 12px|微软雅黑 11px|
|macOS|SF Pro 14px 加粗|SF Pro 12px|SF Pro 11px|
### 4.3 组件样式

1. **按钮**：

    - 圆角：4px；

    - 常规按钮：无边框，hover 背景 `#E0E0E0`；

    - 主要按钮：背景 `#0078D4`，文字白色，hover 背景 `#0066CC`；

2. **输入框/下拉框**：

    - 边框：1px `#E0E0E0`，聚焦 1px `#0078D4`；

    - 圆角：4px；

    - 内边距：8px 12px；

3. **滚动条**：

    - 宽度：8px；

    - 常态：背景透明，滑块 `#CCCCCC`；

    - hover：滑块 `#999999`；

    - 选中：滑块 `#0078D4`。

## 五、适配与兼容性

### 5.1 分辨率适配

- 最低分辨率 1280×720：自动隐藏右侧属性面板，保留核心区域；

- 窗口缩放：面板尺寸按比例自适应，核心组件（按钮/文字）不缩放，保证可读性；

### 5.2 跨平台适配

|差异点|Windows 适配规则|macOS 适配规则|
|---|---|---|
|窗口样式|原生窗口边框，支持最大化/最小化|适配 macOS 圆角窗口，关闭按钮居右|
|快捷键|Ctrl 替换为 Command|Command 替换为 Ctrl（文档标注）|
|字体|微软雅黑|SF Pro|
|交互|右键菜单为主|右键/双指点击均支持|
## 六、性能要求

1. **启动速度**：首次启动 ≤ 3 秒，二次启动 ≤ 1 秒；

2. **文件加载**：1000 个图片文件目录加载 ≤ 5 秒（仅加载缩略图）；

3. **预览速度**：单张 RAW 图片预览 ≤ 1 秒，4K 视频缩略图预览 ≤ 2 秒；

4. **内存占用**：常规使用（≤500 文件）内存占用 ≤ 500MB。
